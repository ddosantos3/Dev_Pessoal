# Barbearia - Landing Page de Agendamento

Este projeto é uma landing page para agendamento de horários em uma barbearia, permitindo ao usuário escolher o profissional, data e horário. Backend em FastAPI, frontend estático (HTML, CSS, JS).

## Estrutura

- Backend: FastAPI, Python 3.11+
- Frontend: HTML, CSS, JS puro (estático)
- Testes: pytest
- Docker/Docker Compose para orquestração
